## Project Overview

This project is divided into two main parts:

1. **Digitizing the archive** of the Berkane Court of First Instance and storing it in an application for easy accessibility from any device connected to the court's server.
2. **Implementing an efficient file circulation system** among court employees to enhance collaboration, workflow efficiency, and to track the lifecycle of each file and its ownership record.

The goal of the project is to improve access to and management of judicial information. This includes developing a web application for archiving and sharing court files.

- Database design and system architecture
- Development of features for importing, managing, and viewing files and documents
- User training on how to use the tool
