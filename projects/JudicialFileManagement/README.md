## Project Overview

This project is divided into two main parts: Firstly, digitizing the archive of the Berkane Court of First Instance and storing it in an application for easy accessibility from any device connected to the court's server. Secondly, implementing an effective file circulation system among court employees to enhance collaboration and workflow efficiency, while also allowing for tracking the lifecycle of each file and its ownership record.
